第二屆前鎮高中競技程式同好會線上賽題解

A. 買鉛筆

直接模擬買3種包裝的情況所需要花費的錢，最後的答案就是最小的花費。
以範例一來說：

  1. 2 2，需要花費58元
  2. 50 30，需要花費60元
  3. 30 27，需要花費54元

所以最後的答案為54

C. 爬山

出處：https://drive.google.com/file/d/1NEsIUWODhSLCyW1TuHIJo5zGj9snQdKR/view
那時候寫到就覺得這題超讚就把它抄過來了XD

用DP求解，定義$dp[i][j]$為從起始點走到$(i,j)$的方法數。
設起始點為$(x,y)$，則$dp[x][y] = 0,dp[x+1][y] = dp[x-1][y] = dp[x][y+1] = dp[x][y-1] = 1$
觀察一下就可以發現走到$(i,j)$的方法數是由$(i-1,j)、(i+1,j)、(i,j-1)、(i,j+1)$這四個方法數湊出來的，要注意的是，只能從低走到高，所以這四個方向只能取高度比他還低的方法數加起來

D. 血緣關係

出處：https://leetcode.com/problems/sum-of-distances-in-tree/

對於兩個相鄰節點A和B，將樹拆成兩個子樹，根節點分別為A和B，A節點到其他所有節點的距離和ans(A) = A子樹中所有節點到A節點的距離和sum(A)+B子樹中所有節點到B節點的距離和sum(B)+B子樹的大小cnt(B)。同理，ans(B) = sum(B)+sum(A)+cnt(A)
由此我們得到：
ans(A) = sum(A) + sum(B) + cnt(B)
ans(B) = sum(B) + sum(A) + cnt(A)
則兩個相鄰節點的解之間的關係為：ans(A) = ans(B) - cnt(A) + cnt(B) = ans(B) - cnt(A) + (N-cnt(A))
因此，對於根節點root的任意子節點child，ans(child) = ans(root)-cnt(child)+N-cnt(child)
得到root的答案就可以DFS得到其他所有節點的答案。(這裡需要一個DFS)
剩下的問題就是解決root的距離和了。
我們一般想到DFS，根節點的距離和$S = \sum s[i] + cnt[i]$，其中s[i]為root的某子節點i到其子節點的距離和，cnt[i]為子節點i的大小。(這裡也需要一個DFS)
